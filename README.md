# Offshore Resource Calculator

A calculator tool for determining offshore recruiting resources needed based on weekly starts targets and conversion rates.

## Features

- Calculate interviews required per week based on starts targets
- Determine number of recruiters needed
- Track recruiting funnel metrics
- Dark mode support
- Mobile responsive
- Teams-optimized view

## Usage

Open `index.html` in a web browser or access via GitHub Pages.

### For Microsoft Teams

Add as a Website tab using the GitHub Pages URL:
```
https://[your-username].github.io/offshore-calculator/
```

Add `?mode=teams` for optimized Teams view.

## Parameters

### Field Recruiting Assumptions
- **Branch Interviews Show Rate**: Percentage of scheduled interviews that attend
- **Passed Interviews Rate**: Percentage of conducted interviews that pass
- **Passed Screenings Rate**: Percentage of passed interviews that pass screening
- **1st Day Arrival Rate**: Percentage of confirmed candidates that show up

### Offshore Assumptions
- **Interviews per Recruiter / Day**: Average interviews scheduled per recruiter per day
- **Onboards per Recruiter / Day**: Alternative KPI - average onboards per recruiter per day
- **Working Days / Week**: Number of working days per week (typically 5)
